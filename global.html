identityFactory(x)

    f(x) = R(callback(x))
or 
    f(x) = R(callback(x()))
if the provided argument is a function.

Aliases

  • value
  • return
  • continueWith

Usage examples

π (pi) as argument to myNextRequestor: 

    var RQ = ('async-rq'),
        rq = ('rq-essentials');

    RQ.sequence([
        rq.value(Math.PI),
        myNextRequestor,
        ...
    ]);

Random numbers as arguments to myNextRequestor: 

    var RQ = ('async-rq'),
        rq = ('rq-essentials');

    RQ.sequence([
        RQ.parallel([
            rq.value(Math.random),
            rq.value(Math.random),
            rq.value(Math.random),
            rq.value(Math.random)
        ]),
        myNextRequestor,
        ...
    ]);

Extra/bonus feature: Including callback argument in string values: 

    var RQ = ('async-rq'),
        rq = ('rq-essentials');

    RQ.sequence([
        rq.value(Math.random),
        rq.value('Sending the random number $(args) to the next requestor'),
        myNextRequestor,
        ...
    ]);

requestorFactory(g)

The primary requestor factory: 

    f(g) = R(callback(g(x)))

Aliases

  • requestorize
  • then
  • do

This requestor factory does not protect/seal/clone incoming arguments in any way. That is the callback function's responsibility!

Usage examples

Transforming a by-value number argument to its rounded version: 

    var RQ = ('async-rq'),
        rq = ('rq-essentials');

    RQ.sequence([
        ...
        rq.then(Math.round),
        ...
    ]);

Transforming a by-reference object argument: 

    var RQ = ('async-rq'),
        rq = ('rq-essentials'),

        obj = {
            val: 13.49
        },
        mutatingAdder = function (obj) { // NB! Mutates incoming arguments
            obj.val += 1;
            return obj;
        },
        pureAdder = function (obj) {
            var newObj = clone(obj);
            newObj.val += 1;
            return newObj;
        };

    RQ.sequence([
        rq.value(obj),
        rq.then(pureAdder),
        ...
    ]);

Parameters:
Name Type Description
g * a function which is applied to the arguments
